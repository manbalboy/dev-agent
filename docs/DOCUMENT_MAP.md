## 3. Maintenance Rule

- 기능을 바꿨다면 최소 아래 셋 중 하나는 같이 갱신한다.
  - README
  - 상위 실행 계획 문서
  - 해당 phase 문서
- 작업 종료 시에는 [CURRENT_HANDOFF.md](./CURRENT_HANDOFF.md)를 같이 갱신한다.
- 문서가 길어져도 snapshot 문서에 최신 구현 세부를 중복해서 적지 않는다.
- 운영 절차는 설계 문서가 아니라 runbook 문서에 둔다.
